My name is Jung-Cheng (Albert) Ho. I have 1.5 year of professional software engineering experience, including experiences in web programming and machine learning.

In April 2020, I joined Micro-System Research Lab at Nation Taiwan University as a Full-time Research Assistant. During my time as a research assistant, I was working writing a C program that enabled our lab's machine learning accelerating chip to complete the training process. Since the chip can only accelerate convolution computations, when it comes to other computations like pooling layer and activation functions, it will need the help from software program. Also, there exist size limitations for convolution computation. Therefore, my program will also help to partition the images into smaller ones. I implemented data reuse method and fused operations to reduce the time consumptions by 100 times. The software + accelerator method achieved the same accuracy (>99%) as that trained by GPU.

Starting from Feburary 2021, I joined Houzz Inc. as a Full-time Full-stack Software Engineer. I worked remotely with the team located in Palo Alto, California. At Houzz, I was familiar with Frontend Development, including React.js, JavaScript, HTML, and CSS. I implemented several changes on home page, including photo carousel, feature lists, and the modification of system message destination. The result of A/B tests shows that the sign-up rate increased significantly. I also worked on a project to migrate and refactor the pricing data retrieval from hard-coded configurations into backend support with GraphQL query. In this project I have the chance to deal with PHP backend and the design of the systems.
